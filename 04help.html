Help

Learning Management System Project

  • Add Task:

    Hier können Sie Aufgaben erstellen, einer Kategorie zuordnen und die Priorität festlegen.

    Optional können bereits hier einer oder mehrere (Ctrl gedrückt halten) User einer Aufgabe zugewiesen werden.

    Sie können die Aufgaben aber auch jederzeit später zuweisen.

  • Backlog:

    Hier können Sie alle Aufgaben und Aktivtäten einsehen, die von ihnen oder ihren Kollegen erstellt wurden.

    Klicken Sie auf eine Aufgabe, um diese zu bearbeiten oder starten Sie das Ticket, indem Sie die Aufgabe an das Board senden.

    Funktionen

    • Hier sehen sie wem die Task zugewiesen ist oder können Tasks zuweisen.
    • Zur schnelleren Zuordnung erkennt man die zugehörige Kategorie.
    • Unter Details sehen Sie den Titel und die Beschreibung.
    • Klicken Sie auf eine Augbage, um diese zu bearbeiten.

  • Board:

    Hier haben sie Ihre zu bearbeitenden Aufgaben im Überblick

    Funktionen

    • Hier können Sie Ihre Aufgaben ganz einfach von einer Liste z.B Todo zur anderen Liste z.B Progress per Drag und Drop verschieben.
    • Wenn Sie auf den Stift drücken, können Sie die Task bearbeiten.
    • Wenn man auf den Mülleimer klickt, werden die Tasks gelöscht.

    Farben

    Zur besseren Übersichtlichkeit haben die verschiedenen Tasks Farben entsprechend der Wichtigkeit.
    • Priority: High = Rot hinterlegt
    • Priority: Medium = Gelb hinterlegt
    • Priority: Low = Grün hinterlegt
