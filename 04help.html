Kanban Project management Tool

Join is a Kanban board. It can help you to give an overview of the current work status and illustrate the project flow in agile project management.


No security layer has been implemented.
This is an example app and should not be used with real data. Do not enter any personal or confidential data!

  • Add Task:

    Here you can create tasks, assign them to a category and set the priority.

    Optionally, one or more (hold Ctrl) users can be assigned to a task.

    But you can always assign the tasks later.

  • Backlog:

    Here you can view all tasks and activities created by you or your colleagues but not yet started.

    Click on a task to edit it or start the ticket by sending the task to the board.

    Features

    • Here you can see to whom the task is assigned or you can assign tasks.
    • For quicker assignment, you can easily recognize the corresponding category.
    • Under Details you can see the title and description.
    • Click on a task to edit it.

  • Board:

    Here you have an overview of your tasks to be processed

    Features

    • Easily move your tasks from one list, e.g. Todo to another list, e.g. Progress by drag and drop.
    • Click the pen icon, to edit the task.
    • Click the trash can icon to delete the task.

    Colors

    For a better overview, the different tasks are color-coded according to their importance.
    • Priority: High = Red
    • Priority: Medium = Yellow
    • Priority: Low = Green
