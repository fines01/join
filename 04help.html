Help

Learning Management System Project

  • Add Task:

    Hier können Sie Aufgaben erstellen, einer Kategorie zuordnen und die Priorität festlegen.

  • Backlog:

    Hier können Sie alle Aufgaben und Aktivtäten einsehen, die von ihnen oder ihren Kollegen erstellt wurden.

    Funktionen

    • Hier sehen sie wem die Task zugewiesen ist.
    • Zur schnelleren Zuordnung erkennt man schnell die zugehörige Kategorie.
    • Unter Details sehen Sie den Titel und die Beschreibung.

  • Board:

    Hier haben sie Ihre Aufgaben im überblick

    Funktionen

    • Hier können Sie Ihre Aufgaben ganz einfach mit Hilfe der Maus von einer Liste z.B Todo zur anderen Liste z.B Progress per Drag und Drop verschieben.
    • Wenn Sie auf den Stift drücken, können Sie die Task bearbeiten.
    • Wenn man auf den Mülleimer klickt, werden die Tasks in Archiv verschoben.

    Farben

    Zur besseren Übersichtlichkeit haben die verschiedenen Tasks Farben entsprechend der Wichtigkeit.
    • Priority: High = Rot hinterlegt
    • Priority: Intermediate = Gelb hinterlegt
    • Priority: Low = Grün hinterlegt
